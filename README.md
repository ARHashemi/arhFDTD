arhFDTD
=======

## C++ package for cumputational electrodynamics based on FDTD method

|               :memo:  Take note of this                             |
|:--------------------------------------------------------------------|
| Documentation and usable examples will be added as soon as possible |


## Requirements
- OpenMP
- OpenCV
- [CImg](https://github.com/dtschump/CImg)

## Build
(require *makedepend*)
```
make depend
make
```
